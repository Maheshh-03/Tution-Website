# Tuition Classes Website

A full-stack tuition classes website built for managing student interactions, authentication, and class-related information.

## Features

- Responsive frontend using HTML, CSS, and Bootstrap
- Backend built with Node.js
- User authentication with JWT
- SQL database integration
- Login and registration system
- Environment variable support using `.env`
- Ready for deployment with backend hosting + database connection

## Tech Stack

### Frontend
- HTML
- CSS
- Bootstrap
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- SQL

### Authentication
- JWT (JSON Web Token)
